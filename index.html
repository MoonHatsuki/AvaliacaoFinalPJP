1- O que é uma publicação web, quais os dois tipos de aplicações existentes e suas principais diferenças.

Se trata de uma aplicação(site) que roda em um operador WEB(google, edge, firefox), os dois tipos de aplicações são:

-Estático: Cujo qual não possui funções dinâmicas e é um site "congelado" ou seja, são apenas informações ou imagens em uma(ou mais) telas, é puramente informativo. EX: Este site, blogs, etc...

-Dinâmico: Possui as mais diferentes funções dinâmicas, desde diversas páginas até animações e funções (EX:Bruno-Simon.com, Agar.io)

2- O que é um domínio e um DNS.

Domínio é o nome que se dá para um site, assim eliminando a necessidade de decorar o IP do site, É exclusivo para o IP do site em específico.
Já o DNS se trata de um sistema de "leitura" dos computadores, revertedo os nomes dos sites para códigos IP, assim facilitando a transferência de dados e garantindo que as requisições sejam enviadas para o local desejado

3- Explique o que é um servidor compartilhado e um servidor dedicado, evidenciando suas vantagens e desvantagens.

O servidor compartilhado é um servidor físico que hospeda diferentes sites de diversas empresas, dividindo os recursos entre si.
O servidor dedicado hospeda apenas o site de uma única organização, assim dedicando todos os seus recursos a ese único site.

4- O que é um servidor web, explique os dois principais servidores. Como fazer o backup de um projeto de forma compactada. Bem como restaurá-la. Utilize imagens para Ilustrar.

Os servidores web são fundamentais para a internet, permitindo acesso fácil a informações e serviços.
Eles podem ser de dois tipos:

Estáticos: Enviam conteúdos que não mudam com frequência, sem precisar processá-los.

Dinâmicos: Processam informações na hora, entregando conteúdo personalizado. Para fazer o backup dos seu projeto de uma forma simples, basta compactá-lo em um único arquivo, (assim facilitando o upload). Compactando no Windows:

2°Selecione "compactar como"

5- O que é um servidor de banco de dados, explique como fazer o backup e restaurar esse backup. Utilize imagens para ilustrar.

Um servidor de banco de dados armazena e gerencia dados, atendendo solicitações de leitura e gravação feitas por aplicações. Exemplos: MySQL, PostgreSQL, SQL Server.
Fazendo o Backup (MySQL):

mysqldump -u usuario -p nome_do_banco > backup.sql

Restaurando:

mysql -u usuario -p nome_do_banco < backup.sql

6- O que é um serviço ssh, e como ele permite acessar um servidor web para transferir arquivos de uma máquina local e o servidor.

É um método seguro de acesso remoto, para utilizá-lo é necessário informar o usuário e o IP da máquina que desejamos acessar
